In this homework, the Food Blog page created during the previous frontend session (Week 8 Lab) was converted into an Express.js application using the Express Generator structure.

The layout, styling, and content of the original Food Blog were preserved exactly while restructuring the project to follow Express and Node.js best practices.

## Technologies Used
- Node.js
- Express.js
- Pug (Template Engine)
- HTML5
- CSS3

## Features
- Express server created using Express Generator
- Static assets (CSS and images) served from the `public` folder
- Pug templates used to render views
- Food Blog layout preserved exactly from the frontend version

## How to Run the Application
1. Clone the repository
2. Navigate to the project directory
3. Install dependencies:npm install
4. Start the server:npm start
5. Open a browser and go to: http://localhost:3000

## Notes
- The `node_modules` folder is excluded using `.gitignore`
- This project follows Express.js best practices and course guidelines
